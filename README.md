## 🚀 ฟีเจอร์ & จุดเด่น

- 🛒 **ระบบจัดการคลังสินค้า (Inventory Management)** - เพิ่ม ลบ แก้ไขข้อมูลสินค้า พร้อมระบบบันทึกเข้าระบบไฟล์
- � **ระบบขายหน้าร้าน (POS System)** - หน้าจอสำหรับเลือกสินค้าลงตะกร้า คำนวณราคารวม พร้อมเสียงแจ้งเตือนเมื่อทำรายการ
- 👥 **ระบบสมาชิก (Membership)** - รองรับการสมัครสมาชิกและจัดการฐานข้อมูลลูกค้า
- 🖨️ **ออกใบเสร็จ PDF (Receipt Generation)** - บันทึกประวัติการขายอัตโนมัติ และสร้างใบเสร็จเป็นไฟล์ PDF ได้ทันที
- 📊 **รายงานยอดขาย (Sales Report)** - ดูสรุปยอดขายสินค้าเพื่อนำไปวิเคราะห์ได้อย่างเป็นระบบ
- 🎨 **Modern GUI** - หน้าต่างโปรแกรมออกแบบให้ทันสมัยและใช้งานง่ายด้วย `CustomTkinter`

## 📦 การติดตั้งโมดูลที่จำเป็น (Required Modules)

### 🚀 รันคำสั่งนี้เพื่อติดตั้งทุกโมดูลในครั้งเดียว!
```bash
pip install customtkinter Pillow fpdf playsound
```

### 📌 คำอธิบายโมดูล (หากต้องการติดตั้งทีละอัน)
- **customtkinter**: `pip install customtkinter` (หน้าตาโปรแกรมสุดล้ำ UI สวยๆ)
- **Pillow**: `pip install Pillow` (ไว้จัดการรูปภาพต่างๆ ภายในแอป เช่น โลโก้, ไอคอน)
- **fpdf**: `pip install fpdf` (ไว้ใช้สร้างและพิมพ์ใบเสร็จออกมาเป็นไฟล์ PDF)
- **playsound**: `pip install playsound` (ไว้เล่นเสียงแจ้งเตือนและเสียงตอนจ่ายเงิน)
